## Installation

1. Install **Sine** if you don’t already have it: https://github.com/CosmoCreeper/Sine
2. Open **Settings → Sine Mods** (or **Cosine**)
3. Search for **Zen Folder Tree Connectors**
4. Install directly from the marketplace!
